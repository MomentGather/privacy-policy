隐私政策 / Privacy Policy

信息收集与使用

为了提供和改进我们的服务，我们可能需要收集以下类型的信息：

  • 用户地址信息：当用户允许我们访问其地址信息时，我们使用该信息来搜索并展示附近的活动。用户提供此权限是可选的。
  • 推送通知：用户可以选择接收来自MomentGather的推送通知，以便在应用关闭时仍能接收到新消息的推送。此权限同样是可选的。
  • 照片库访问：我们请求访问用户的照片库，以便用户可以选择图片作为用户头像、照片墙或活动照片。用户授予此权限是可选的。

Information Collection and Use

To provide and improve our services, we may collect the following types of information:

  • Location Information: We use location information to search for and display events nearby when users grant us access to their location. Granting this permission is optional.
  • Push Notifications: Users may opt to receive push notifications from MomentGather to get updates on new messages even when the app is closed. This permission is also optional.
  • Photo Library Access: We request access to users' photo libraries so that they can select pictures as their user avatar, for a photo wall, or as event photos. Granting this permission is optional.

数据存储

用户创建的个人资料和活动数据将被存储在我们的数据库中，我们采取适当的安全措施来保护存储的信息不被未经授权的访问、披露、修改或销毁。

Data Storage

Personal profiles and activity data created by users are stored in our database. We take appropriate security measures to protect stored information against unauthorized access, disclosure, alteration, or destruction.

错误收集

我们使用第三方服务Sentry来收集应用的报错信息，以帮助我们诊断问题并改善应用性能。收集的信息仅用于内部分析。

Error Collection

We use the third-party service Sentry to collect error information from the app, which helps us diagnose problems and improve app performance. Collected information is used solely for internal analysis.
